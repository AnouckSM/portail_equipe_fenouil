# Portail équipe — Equipe Fenouil

Ce dépôt contient la version statique du portail équipe (Projet Renov'EA - Urssaf ACOSS).

Vous êtes l'unique éditrice du fichier `index.html`. Pour mettre à jour le site : éditez `index.html`, commit & push.

Déploiement rapide :
- Netlify Drop : glisser-déposer `index.html` pour publication immédiate (URL stable fournie).
- Netlify connecté : connecter ce repo à Netlify pour déploiements automatiques à chaque push.
- GitHub Pages : activer Pages sur la branche `main` et déployer depuis la racine.

Fichiers inclus :
- index.html  — la page principale
- netlify.toml — configuration minimale Netlify
- .github/PULL_REQUEST_TEMPLATE.md — template PR
- .github/workflows/deploy-pages.yml — workflow optionnel pour GitHub Pages

Sécurité :
- Ne publiez pas de données sensibles dans la page.
- Si le contenu doit rester restreint, utilisez un repo privé et Netlify connecté avec contrôle d'accès, ou n'utilisez pas Netlify Drop.
